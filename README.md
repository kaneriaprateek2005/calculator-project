Calculator Project

About

This is a basic calculator made using HTML, CSS and JavaScript.
It can perform simple calculations like addition, subtraction, multiplication and division.

I created this project to practice JavaScript concepts and improve my frontend skills.

---

Features

- Perform basic arithmetic operations
- Simple and clean user interface
- Works on browser without any installation

---

Files in this project

- "index2.html" → main structure
- "style2.css" → design and layout
- "script2.js" → calculator logic

---

How to use

1. Download or clone this repository
2. Open "index2.html" in your browser
3. Start using the calculator

What I learned

- Working with DOM
- Basic UI design using CSS
